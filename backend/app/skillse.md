# Adaptive Learning Final Draft Skill

Use the initial curriculum draft, the diagnosis report (which highlights the user's weaknesses and strengths), and the log of user answers to refine and finalize the curriculum.

## Guidelines
1. Ensure every module has specific, actionable steps.
2. Incorporate resources matching the user's previously provided knowledge base.
3. If the user failed certain prerequisite checks (from the diagnosis report), add foundational modules at the beginning of the roadmap to address them.
4. Keep the output highly structured, easy to parse, and formatted cleanly.
5. Maximize relevance to the user's goals.
